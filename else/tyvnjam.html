I've been added! Now what?

Add the widget to your website using the following code:

<div id='tyvnjam'><script type="text/javascript" src="https://www.kazehai.com/else/tyvnjam/variables.js"></script><script type="text/javascript" src="https://www.kazehai.com/else/tyvnjam/widget.js"></script></div>

Place this code in the <head> section of the page containing the widget:

<link rel="stylesheet" href="https://www.kazehai.com/css/tyvnjam.css">
